Commands

Variables are indicated with {what-to-fill-in}, optional variables with [what-to-fill-in]

Command: Description: Required permission(s):
>help Shows a help menu None
>guildid (>guild-id) Gives the guild ID from the server None
>serverinfo (>server-info) Gives info about the server None
>myinfo Gives info about you None
>info {@user} Gives info about {@user} None
>invite Says an invite for Utilly None
>invitestudio (>is) [MaxUses] [Age] Creates an invite to the current channel None
>ping Gives Utilly's latency in ms None
>leave Leave the current server None
>channel-info (>ci, >channelinfo) Gives information about the current channel None
>kick {@user} [reason] Kicks {@user} for [reason] kick_members
>ban {@user} [reason] Bans {@user} for [reason] ban_members
>purge (>clear) [amount] Clears [amount] messages manage_messages
>nuke Clears all messages in a channel* manage_messages
>slowmode {amount} Sets the slowmode to {amount} seconds, 0 to turn it off manage_channels

*Messages older then 2 weeks can't be deleted because of the Discord API
